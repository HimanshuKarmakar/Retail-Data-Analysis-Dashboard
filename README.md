# Retail-Data-Analysis-Dashboard
Retail Performance Intelligence Dashboard built in Excel using POS/ERP data to analyze sales, returns, and customer behavior. Features KPI cards, pivot-based insights, weekday vs weekend trends, and actionable recommendations for discount strategy, product mix, and profitability optimization.

## Dataset Used
- <a href="https://docs.google.com/spreadsheets/d/1GzcfjCPA5nrTCu1bBHaTfYt8nTKqPyqkGq__OEuC9kw/edit?usp=sharing"> Dataset</a>

## Questions(KPIs)
- Which states generate the highest and lowest total sales?
- How does total sales distribution vary across sales channels (In-Store, Online Delivery, Self-Checkout)?
- Which product categories contribute the most to total revenue in each state?
- Which items generate the highest total sales?
- Which items have the highest number of returns?
- Is there a relationship between high-selling items and high return counts?
- Which products record the highest quantity sold?
- How does quantity sold vary across different items?
- How does category-wise revenue distribution differ across states?
- Which sales channel contributes the maximum share of overall revenue?

## Process Followed
- Generated structured retail transaction data for analysis.
- Cleaned and formatted raw data using Excel and Power Query.
- Created calculated fields such as sales, profit, and discount buckets.
- Built pivot tables to compute and analyze key KPIs.
- Generated charts for trend analysis and performance comparison.
- Designed an interactive dashboard using slicers.
- Interpreted insights to support business decision-making.
